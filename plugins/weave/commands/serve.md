---
description: launch the live weave workspace in a browser
argument-hint: [optional --port N]
---

Launch the weave workspace UI: `weave serve $ARGUMENTS`. Defaults to port 4747 and opens a browser tab. Use `--no-open` to skip the browser launch (e.g. when running on a remote host). The workspace lets the human edit nodes, attach files, set focus, undo edits, run git commits and click any file node to read it inline.
